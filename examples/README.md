# Examples

These examples show how to use `@stackline/loading` in real product surfaces without depending on a framework wrapper.

- [`dashboard-card.ts`](./vanilla/dashboard-card.ts): card overlay for KPI refresh
- [`button-loading.ts`](./vanilla/button-loading.ts): inline button state while a form submits
- [`container-overlay.ts`](./vanilla/container-overlay.ts): table or section loading
- [`fullscreen.ts`](./vanilla/fullscreen.ts): fullscreen application shell loading
- [`upload-zone.ts`](./vanilla/upload-zone.ts): upload area feedback
- [`anti-flicker.ts`](./vanilla/anti-flicker.ts): delay and minimum visibility pattern
